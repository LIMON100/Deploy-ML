# Maixduino

### Download 
    Maixpy IDE
    KFLASH_GUI



1. Flash the board

2. Download micorpython firmware (https://dl.sipeed.com/shareURL/MAIX/MaixPy/release/master/maixpy_v0.6.2_72_g22a8555b5)

3. Configure with kFlash

4. Go to maixpy Ide -> Tools -> select-board -> Select Sipped Maixduino
